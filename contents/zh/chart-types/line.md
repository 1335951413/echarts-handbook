# 柱状图

柱状通过柱形的高度来表现数据的大小，用于有至少一个类目轴或时间轴的直角坐标系上。

## 最简单的柱状图

设置柱状图的方式，是将 `series.name` 设为 `"bar"`。最简单的柱状图可以这样设置：

```js
option = {
    xAxis: {
        data: ['周一', '周二', '周三', '周四', '周五', '周六', '周日']
    },
    yAxis: {},
    series: [{
        type: 'bar',
        data: [23, 24, 18, 25, 27, 28, 25]
    }]
};
```
